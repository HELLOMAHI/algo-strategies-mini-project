## Overview
This project demonstrates the use of different **algorithmic strategies** to solve real-world problems.  
Each problem is implemented in Python with **time and memory profiling** and **performance visualization**.

---

## Problems Implemented

| Problem | Algorithmic Strategy | Domain | Objective |
|----------|----------------------|---------|------------|
| TV Commercial Scheduling | Greedy (Job Sequencing) | Media & Advertisement | Schedule ads to maximize total revenue |
| Budget Maximization | Dynamic Programming (0/1 Knapsack) | Budget / Investment | Select items/projects for maximum profit under a budget |
| Sudoku Solver | Backtracking | Gaming / Puzzle | Fill Sudoku grid under constraints |
| Password Cracking | Brute Force | Cybersecurity | Try all combinations to find the correct password |
